# react-native-mcemoji-picker

An iOS emoji-picker component for  react-native

## Installation

```sh
npm install react-native-mcemoji-picker
```

## Usage

```js
import { multiply } from 'react-native-mcemoji-picker';

// ...

const result = await multiply(3, 7);
```
